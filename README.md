# BJAC Playwright Automation Framework

A robust end-to-end UI automation framework developed using Playwright with JavaScript.

## Features

- Playwright Test Runner
- Page Object Model (POM)
- Multi-Environment Support (.env)
- Data-Driven Testing (CSV & JSON)
- Allure Reporting
- HTML Reporting
- Reusable Utilities
- Logging
- Cross-Browser Execution
- CI/CD Ready
- Screenshot & Trace Capture

## Tech Stack

- Playwright
- JavaScript
- Node.js
- Allure Report
- CSV Parser
- Dotenv

## Supported Modules

- Bidder Registration
- Consignment
- Vehicle Management
- Account Creation
- Login & Authentication
- End-to-End UI Workflows
